# homebrew-tap

A [Homebrew](https://brew.sh) tap for [rustutils](https://github.com/rustutils) — a collection of Unix utilities reimplemented in Rust.

## Install

```sh
brew tap rustutils/tap
```

Then install any formula:

```sh
brew install rustutils/tap/git-lfs
```

## Available formulae

| Formula | Description |
| --- | --- |
| [`git-lfs`](Formula/git-lfs.rb) | Git LFS — large file storage for git, reimplemented in Rust |

## Tapping from GitLab

If you'd rather pull the tap from GitLab:

```sh
brew tap rustutils/tap https://gitlab.com/rustutils/homebrew-tap.git
```
